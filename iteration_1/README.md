# Sign Language Detector (Iteration 1)

This is a standalone package for the Sign Language Detector.

## Setup

1.  Install Python 3.9+ (Tested with 3.12).
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Running

Run the application:

```bash
python main.py
```

## Controls
- **A-Z**: Switch target letter (Static mode for A-Y, Dynamic mode for J/Z).
- **SPACE**: Start recording (only for dynamic letters J/Z).
- **ESC**: Quit.
